# GREwords

### 此项目为前后端分离项目，旨在提供公开的GRE单词背诵网站。

用户可以访问网站进行单词背诵，同时网站会收集用户信息进行统计，帮助用户更快记住单词

### 需求

1. 用户信息： 名字，单词背诵量，错最多的单词，个性化单词表
2. 单词背诵卡片（单词名，翻译，解释，例句），单词分组记忆，单词混合记忆
